# rodrigoangelcollazo.github.io

Personal landing page for **Rodrigo Angel Collazo** (ML Engineer, Buenos Aires, Argentina).

Main profile: `https://github.com/Rodrigoangelcollazo`

## Run locally

- **Option A (Python)**:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

- **Option B (Node)**:

```bash
npx serve .
```

## Deploy

This repo is designed for **GitHub Pages**. Push to `main` and enable Pages for the repository.
